tiny + line pixel drawing.
a colorful refresh of Bresenham's line algorithm.
click and drag with the mouse to draw.
resolution and color change by pressing numbers 1 to 9.
resolution and number are inversely proportional. color will change randomly.
random sequence of colors will flash as you draw if number stays pressed.
press number 0 to bleach the canvas and spacebar to clear.

To view this content, you need to install Java from java.com
